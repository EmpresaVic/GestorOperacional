🔒 Informações sobre Armazenamento de Dados (GitHub Pages)
  • Armazenamento local: Todos os dados são salvos no localStorage deste navegador, neste dispositivo. Dados não são enviados para nenhum servidor.
  • Sem sincronização: Cada dispositivo tem sua própria cópia dos dados. Para compartilhar, use a função Exportar/Importar JSON.
  • Segurança do repositório: Mantenha o repositório GitHub como Privado se os dados forem internos da empresa.
  • Backup recomendado: Exporte regularmente os dados em JSON e armazene em local seguro (nunca versionado no repositório público).

⚠️ Aviso de Segurança — Gestão de Usuários no GitHub Pages
  • O sistema de login é client-side: as senhas ficam armazenadas com hash simples no localStorage do navegador.
  • Não use senhas corporativas reais aqui. Crie senhas exclusivas para este sistema.
  • Qualquer pessoa com acesso ao dispositivo pode inspecionar o localStorage via console do navegador.
  • Para ambientes com dados sensíveis, considere migrar para uma solução com backend autenticado (ex.: Firebase Auth + Firestore).
  • Altere a senha padrão admin/1234 imediatamente após o primeiro uso.
